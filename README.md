## Troubleshooting

**No media sources detected**
Make sure the video page is fully loaded and playback has started. The iframe player handoff needs to resolve before sources become visible.

**The popup shows no options**
Try refreshing the page and starting the video again. Some Pornmedium pages may require a second load for the handoff to complete.

**Download starts but fails partway**
Check your internet connection. Large m3u8 streams can be interrupted by network drops.

**The extension icon is greyed out**
You may be on a page that does not match the supported slug-plus-ID route pattern. Navigate to a valid Pornmedium video page.

**Sources appear but do not match the video**
The iframe handoff can sometimes expose multiple streams. Try selecting a different source option if one is available.
